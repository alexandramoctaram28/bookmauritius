# bookmauritius

A simple static hotel reservation page for Mauritius-only bookings.

Features:
- Mauritius hotel reservation form
- WhatsApp reservation and payment contact: +230 58023676
- Booking email: bookmauritius@outlook.com
- Automatic commission calculation per booking
- Public static hosting ready (works with Netlify, GitHub Pages, or any static web host)

Local preview:
- Open http://127.0.0.1:8000/index.html

Public deployment:
- Push this repository to GitHub
- Connect it to Netlify or GitHub Pages
- Use the included netlify.toml for routing
